# TokyoLED Night - Omarchy Theme

An Omarchy Theme for your Arch Linux / Hyprland setup.

A high-contrast, OLED-optimized theme with deep blacks and sharp accents. Designed for minimalist efficiency and battery savings on OLED displays.

## OLED Tokyo Night Palette (Official Night Style)

- **Background:** `#000000` (Pure Black OLED)
- **Foreground:** `#ffffff` (Pure White)
- **Selection:** `#33467c`

### Accents (Normal)
- **Red:** `#f7768e`
- **Green:** `#9ece6a`
- **Yellow:** `#e0af68`
